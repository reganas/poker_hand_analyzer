Poker Hand Analyzer

Overview
This project simulates a poker hand evaluator that evaluates the hands, evaluates win probabilities as per user's input.

Features
- Hand Evaluation
- Win Probability Estimation
- Opponent Hand simulation
- Hand history
- Interactive CLI for user input
